# 使用说明
```
1. 在根目录执行 `npm install`
```
## 安装全部依赖

```
2. npm run examples:install
```
## 一键启动主应用与子应用

```
3. npm run examples:start
```

> **注意：**

> 主应用端口默认：`8080`

> 通信方案采用官方提供的`initGlobalState`方式

> 当网络较差时，安装依赖可能会失败，这时可以进入对应的项目单独安装，查看对应的`pack.json`即可
